# 📊 Dashboard

This project includes an interactive Power BI dashboard for analyzing global military power.

---

## 🚨 Note

Due to file size limitations, the `.pbix` file is not uploaded to GitHub.

---

## 📊 Dashboard Features

### 1. Quick Stats

* Overview of global rankings
* Key KPIs and trends

### 2. Nation Overview

* Detailed military profile of each country

### 3. Compare Powers

* Side-by-side comparison of two countries

### 4. Coalition Builder

* Analyze combined strength of multiple nations

---



## 🛠️ Tools Used

* Power BI
* Python (Data Processing)

---

## 📝 Description

The dashboard visualizes military and economic indicators for 140+ countries, enabling comparison, analysis, and strategic insights.
