# 🌍 Unified Military Analytics and Comparison Dashboard (2025)

## 📌 Project Overview

This project is an end-to-end data analytics solution designed to analyze and compare global military power across 140+ countries using more than 50 defense, economic, and infrastructure indicators.

The system integrates web scraping, data cleaning, KPI engineering, and visualization to provide meaningful insights into global military capabilities.

---

## 🎯 Objectives

* Scrape military and economic data from GlobalFirepower
* Clean and standardize raw data
* Engineer key performance indicators (KPIs)
* Enable comparative analysis across countries
* Prepare data for interactive dashboards

---

## 🧠 Key Features

* 📊 Analysis of 140+ countries
* 📈 50+ military and economic indicators
* 🔍 KPI-driven insights
* 🌐 Cross-country comparison
* 🧩 Modular and scalable pipeline

## 🔄 Project Workflow

1. **Web Scraping**

   * Extracted data from multiple GlobalFirepower pages
   * Collected country-wise metrics

2. **Data Cleaning**

   * Removed special characters and inconsistencies
   * Converted values into numeric formats
   * Handled missing values

3. **Data Integration**

   * Merged multiple datasets into a unified structure
   * Ensured consistency across all countries

4. **KPI Engineering**

   * Assets per Capita
   * Budget-to-GDP Ratio
   * Power Index Rank Gap

5. **Analysis & Insights**

   * Identified top-performing countries
   * Compared military capabilities across regions

### Key Data Categories:

* Military manpower
* Airpower & naval strength
* Land systems (tanks, artillery)
* Economic indicators (GDP, budget, debt)
* Infrastructure (airports, railways, ports)

---

## 📈 Sample Insights

* The United States, China, and Russia dominate global military strength
* Population size influences manpower but not efficiency
* Smaller countries often have higher assets per capita
* Naval strength depends heavily on geography
