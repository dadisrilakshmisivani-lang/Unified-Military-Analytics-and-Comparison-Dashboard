This folder contains all the data resources used in the **Unified Military Analytics and Comparison Dashboard** project.

---

## 📌 Contents

### 1. 📄 Problem Statement

* File: `DV_Unified Military Analytics and Comparison Dashboard.pdf`
* Description: Contains the complete project requirements, objectives, and workflow.

This project focuses on analyzing global military power for 140+ countries using scraped data and building interactive dashboards.

Key highlights:

* Analysis of 50+ military and economic indicators
* Comparison of countries and coalition strength
* KPI metrics like Power Index Rank Gap and Budget-to-GDP ratio 

---

### 2. 🔗 Data Sources

* File: `links for military data.txt`
* Description: Contains all the URLs used for web scraping.

The data is collected from **GlobalFirepower.com**, covering:

* Population and manpower
* Air, land, and naval assets
* Defense budgets and economic indicators

Example metrics:

* Total military aircraft
* Active personnel
* Tanks and naval fleet
* Defense budget and GDP-related metrics 

---

### 3. 📊 Raw Data

* Folder: `raw/`
* Description: Contains original datasets collected through web scraping.

⚠️ Important:

* Raw data is **not modified**
* Used as the base for cleaning and analysis


---

## 🔄 Data Workflow

1. Web Scraping → Extract data from provided URLs
2. Raw Data Storage → Store original scraped data
3. Data Cleaning → Process and structure data
4. Analysis & Visualization → Build dashboards

---

## 📝 Notes

* All data is collected from publicly available sources.
* Cleaned datasets will be stored in the `cleaned/` folder.
* This folder ensures reproducibility and transparency of the project.
